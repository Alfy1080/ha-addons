Honeygain Add-on

About

This add-on hosts Honeygain directly inside your Home Assistant instance.

Honeygain is an application that allows you to earn passive income by sharing your unused internet connection. The application securely handles your background traffic without accessing your personal data.

Features

Secure Configuration: Keeps your passwords out of configuration files and utilizes the Home Assistant UI for secure secret storage.

Custom Naming: Easily set a custom device name so you can identify your Home Assistant instance within the Honeygain web dashboard.

Lightweight: Uses a minimal base to ensure low overhead on your Home Assistant OS.

Please read the Documentation tab for setup instructions.
