Maintainerr Add-on

About

This add-on hosts Maintainerr directly inside your Home Assistant instance.

Maintainerr is an application that looks and smells like Overseerr, but does the opposite. It acts as a maintenance tool for the Plex ecosystem, allowing you to define rules for cleaning up old or watched media to free up disk space.

Features

Integrated Storage: Automatically uses Home Assistant's persistent storage for your database and rules.

Shared Access: Automatically maps your Home Assistant /share and /media mounts so you can point it right at your local or network storage.

Configurable: Tweak timezones, debug modes, and reverse-proxy base paths directly through the Home Assistant UI.

Please read the Documentation tab for setup instructions.
